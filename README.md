# Онлайн библиотека

### Основные возможности
Управление книгами - полный CRUD для книг, авторов, жанров
Читательские билеты - система учета читателей и выданных книг
Двойная аутентификация - OTP-верификация для защищенных операций
Умные фильтры - поиск по авторам, жанрам, годам издания
Административный контроль - разделение прав доступа для сотрудников

### Технологический стек
Backend: Django REST Framework, PyOTP
Frontend: Vue 3, Pinia (state management), Axios, Bootstrap 5

### Особенности безопасности
Сессионная аутентификация
Двухфакторная OTP-верификация для операций изменения
Разделение прав между обычными пользователями и администраторами

### UI
Адаптивный дизайн с Bootstrap 5
Модальные окна для редактирования
Динамическая фильтрация без перезагрузки страницы
Интуитивное управление книгами и читателями
