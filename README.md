# Онлайн библиотека

### Основные возможности
-  Управление книгами - полный CRUD для книг, авторов, жанров
-  Выданные книги - система учета читателей и выданных книг
-  Фильтры - поиск по авторам, жанрам, годам издания
-  Административный контроль - разделение прав доступа для админов и читателей

### Стек
-  Backend: Django REST Framework, PyOTP
-  Frontend: Vue 3, Pinia (state management), Axios, Bootstrap 5

### Безопасность
-  Сессионная аутентификация
-  Двухфакторная OTP-верификация для операций удаления
-  Разделение прав между обычными пользователями и администраторами

### UI
Адаптивный дизайн с Bootstrap 5
Модальные окна для редактирования
Динамическая фильтрация без перезагрузки страницы
